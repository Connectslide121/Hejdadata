A privacy-first web application that helps Swedish residents easily request removal of their personal information from public directory services according to GDPR.

**Visit site:** [https://hejdadata.se](https://hejdadata.se)

## Features

‚úÖ **100% Privacy-First** - All processing happens in your browser. No data is stored or sent to any server.

‚úÖ **GDPR Compliant** - Pre-formatted email templates based on GDPR Articles 17 & 18.

‚úÖ **10 Swedish Directory Services** - Covers major platforms including MrKoll, Ratsit, Hitta, Eniro, Merinfo, Upplysning, Birthday, Biluppgifter, Lexbase, and Krimfup.

‚úÖ **Bilingual Support** - Full support for Swedish and English.

‚úÖ **No Registration Required** - Completely free, no sign-up, no tracking.

‚úÖ **Modern UI** - Clean, responsive design with smooth animations and glassmorphic effects.

## How It Works

1. **Enter Your Details** - Provide your name, email, and optionally your personal identity number (personnummer).
2. **Generate Requests** - Automatically creates personalized GDPR deletion requests for all 10 services.
3. **Send Emails** - Click individual mailto links to open pre-filled emails in your email client and send them yourself.

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Icons**: [Lucide Icons](https://lucide.dev)
- **Logos**: Clearbit Logo API + Custom assets
- **Hosting**: GitHub Pages

## Privacy & Security

- **No Backend** - Entirely client-side application
- **No Data Storage** - Nothing is saved to cookies, localStorage, or any database
- **No Analytics** - No tracking or telemetry of any kind
- **No External API Calls** - Except for logo fetching (Clearbit)
- **Open Source** - Full transparency, inspect the code yourself
